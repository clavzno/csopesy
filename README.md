TO RUN THE PROGRAM
1. From this root folder, do `cd csopesy` twice or navigate to csopesy(root)\csopesy\csopesy\csopesy.cpp
2. Run `g++ csopesy.cpp -o csopesy.exe`
3. Run `./csopesy` or open `csopesy.exe` in the same folder OR click the run button
4. Follow on-screen instructions

(updated 10/14/2025) EDITING CODE
1. Clone the Github repository to your local device. It would be in /C:/Users/<username>/source/repos/<csopesy folder>
2. Open the local repository in Visual Studio
3. Within the csopesy folder, open `csopesy.sln` (Solution File)
4. Navigate to "Source Files" and open `csopesy.cpp`
